Bear's math adventure:
This is parts of the game I created for my thesis which's idea was to encourage children around 5th and 6th grade to learn more math by making it more fun and while most of the code and scenes are added in here the Unity store assets are not as I do not have rights to distribute them freely, they were free assets anyway so they are easy to find in the store. I shall keep the apk file in store in case someone wants to give it a try.

The game was formed of couple scenes that acted as levels, each scene was made up of a Zone that contained general information for which level it was and such and then it acted as a parent for the Cell objects which were used to form randomized map through which player had to find their way. Each Cell had a tile that contains the landscape for the map and objects that controlled tasks and enemies within the Cell that the player had to defeat.
